A paragraph is an example of block-level element.

Block level elements are displayed in a new line and they take 100% of the width.

Anchor tag is an example of inline element - Click here to get to the GOOGLE. Inline elements have their width defined by the width of the content they hold. They don't get displayed in a new line and they don't force any other elemnt to get display in a new line.

Image tag is also an example of inline element. This is not my last paragraph.

Inline elements should always be contained by a block level element
